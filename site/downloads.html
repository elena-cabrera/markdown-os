Download Markdown-OS

A native desktop app for your markdown files. No browser required — runs locally with the same powerful editor.

Detected: macOS

macOS

Download .dmg
Markdown-OS-macos.dmg

System Requirements

  • macOS 12 (Monterey) or later
  • Apple Silicon (M1+) or Intel
  • ~80 MB disk space

Installation

  1. Open the downloaded .dmg file
  2. Drag Markdown-OS to your Applications folder
  3. On first launch, right-click the app and select "Open" (required for unsigned apps)
  4. Or go to System Settings → Privacy & Security → "Open Anyway"
Detected: Windows

Windows

Download .zip
Markdown-OS-windows.zip

System Requirements

  • Windows 10 (1809) or later
  • x64 processor
  • ~100 MB disk space

Installation

  1. Extract the downloaded .zip file
  2. Open the extracted folder
  3. Run Markdown-OS.exe
  4. If SmartScreen warns about an unrecognized app, click "More info" → "Run anyway"

About code signing

These builds are not currently code-signed. macOS and Windows may show security warnings on first launch. The app is open-source — you can inspect the source code and build from source if you prefer.

Prefer the command line?

Install via pip or uv and run from your terminal.

uv tool install markdown-os
pip install markdown-os
